Missing Cards
Cards that couldn't be found during pulling. Scan to re-shelve, or mark as gone.
At Grading
Cards physically sent out for grading. Scan a barcode or click Reverse to send the card to the Return Queue for re-shelving.
Audit
Pick a bin or binder, scan everything in it. Wrong-bin and unknown barcodes flag in real time. Anything left at the end can be marked Missing.
Game Tags
Cards whose stored game disagrees with what their set resolves to in Scrydex — the routing landmines. One-click fix corrects the tag (and re-routes to the Return Queue if it's sitting in the wrong kind of bin).
Return Queue
Cards pending return to storage
Display Case
What's currently on the customer-facing glass. Phase-1 shell — Set Out / Return flows ship next.
Binders
Customer-browse binders. Fill from storage with the same scoring algorithm; pull cards out via scan flow.
Sell Queue
Scan cards from the binders or display case. We create a Shopify draft listing per card with the barcode as the SKU — the POS device then scans the same SKU to ring it up.
Featured Cards
Cards that get a scoring boost when picking what to put in the display case. Patterns are case-insensitive substrings — "Charizard" matches Dark / Shining / Reverse / etc.
Card Editor
Search any raw card in stock and repair its identity, condition, variant, or price. Scan a barcode to jump straight to that card's edit panel.
